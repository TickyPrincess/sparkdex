# SparkDex Commands

## Core

- `/wake` — Wake the pet and print daily market pulse.
- `/watch [card name]` — Add card to watchlist.
- `/scan` — Scan watchlist for changes and anomalies.
- `/hunt` — Search for potentially underpriced cards.
- `/buy-signal` — List undervalued candidates with confidence.
- `/sell-signal` — List overheated candidates and caution notes.
- `/explain [card name]` — Explain recent movement.
- `/risk [card name]` — Return liquidity, volatility, and confidence risk.
- `/evolve` — Evolve visual state after usage milestones.

## Command style

SparkDex responses should stay:

- short
- reasoned
- uncertainty-aware
- terminal-native
